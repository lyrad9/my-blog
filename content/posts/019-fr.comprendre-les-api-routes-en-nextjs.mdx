Les API Routes sont pour la pluspart un concept en pogrammation difficile à assimiler lorsqu'on est débutant. En NextJS, tu as la possibilité de créer des API Routes qui vont te
permettre de gérer des requêtes serveur et de fournir des données aux composants de votre aplication.  Dans cet article, je vais explorer avec toi les API Routes 
plus précisement **les Routes Handlers** et te motrer comment les utiliser pour faire communiquer.

## C'est quoi une API?
une <Code>API</Code> (Application Programming Interface) est un ensemble de règles, protocoles et outis qui permettent à différetes applications de communiquer entre elles. 

## C'est quoi une API Route?

Une API Route est une partie spécifique d'une API qui gère une requête <Code>HTTP</Code> spécifique
